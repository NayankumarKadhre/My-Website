Task Inventory App

Designed and developed a Task Inventory web application using Node.js, Express.js, EJS and MongoDB having a clean and intuitive interface to optimize the user experience. Implemented features that allow users to effortlessly add tasks, define priority, difficulty levels, and time period, with storage in a MongoDB database. Also added task deletion functionality for users to easily manage their tasks. Leveraged the power of MongoDB Atlas, ensuring scalable and globally accessible data storage for enhanced performance and reliability.

Hindi Handwritten Digit Recogniton App

In this project, a Hindi Handwritten Digit Recognition application has been developed, which provides the users with an intuitive interface to upload photos of handwritten digits or draw digits directly on a canvas for prediction. Leveraging our custom-built hybrid CNN-LSTM model with an accuracy of 99.98%, the application accurately predicts the corresponding digit label with high precision.
